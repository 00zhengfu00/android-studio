# 方法间前后跳转

## 跳转到下一个方法

### 操作步骤:

菜单栏: Navigate —&gt; Next Method\(下一个方法\)

### 快捷键:

Mac: control + ↑ \(此快捷键与可能与系统快捷键冲突,请自行修攺\)

Windows\/Linux: Alt + ↑

## 跳转到上一个方法

### 操作步骤:

菜单栏: Navigate —&gt; Previous Method\(上一个方法\)

### 快捷键:

Mac: control + ↓ \(此快捷键与可能与系统快捷键冲突,请自行修攺\)

Windows\/Linux: Alt + ↓
