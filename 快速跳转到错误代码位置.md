# 快速跳转到错误代码位置

代码如果出现错误,编辑器会在右边栏高亮显示红色的条标,如果有多处错误就会显示多个条标.

如果想快捷查看错误代码,可以通过点击右边栏的条标进行快速跳转。

跳转到下一个错误位置

菜单栏: Navigate —&gt; Next Highlighted Error

快捷键:

Mac: Fn + F2

Windows\/Linux: F2

跳转到上一个错误位置

操作步骤: 

菜单栏: Navigate —&gt; Previous Highlighted Error

快捷键:

Mac: Fn + Shift + F2

Windows\/Linux: Shift + F2
