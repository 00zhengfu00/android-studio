# 管理书签

对书签进行搜索、排序、修攺描述、删除、查看等操作

## 打开书签管理界面

### 操作步骤:

菜单栏: Navigate —&gt; Bookmarks —&gt; Show Bookmarks

### 快捷键：

Mac: Fn + Command + F3

Windows\/Linux: Shift + F11

## 修攺书签描述

给书签加上描述信息

### 操作步骤:

 在书签管理界面点击左上角的编辑按钮 —&gt; 在弹窗中输入描述

### 

快捷键：

Mac: Command + 回车 —&gt; 在弹窗中输入描述

Windows\/Linux: Shift + F11
