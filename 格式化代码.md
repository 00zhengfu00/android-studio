# 格式化代码

### 操作步骤:

菜单栏: Code —&gt; Reformat Code

右键菜单: Reformat Code

### 快捷键:

Mac: option + command + L

Windows\/Linux: Ctrl + Alt + L



## 配置格式化选项



通过快捷键\(Mac\): option + command + shift + L 打开配置对话框



## 自动缩进行

### 操作步骤:



菜单栏: Code —&gt; Auto-Indent Lines



### 快捷键: 

Mac: option + control + I 

Windows\/Linux: Ctrl + Alt + I
